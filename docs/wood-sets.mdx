## Dyed Wood Sets


Added in Pyrite 0.2, Pyrite includes a wood set for each of the vanilla Dyes. These sets include planks, slabs, stairs, crafting tables, fences, fence gates, doors, trapdoors, pressure plates, signs, hanging signs, and buttons.
